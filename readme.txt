== Description ==

This plugin adds a floating button to the website with the aim of prompting the user to start a WhatsApp conversation with the website owner.

Please report any bugs to plugins in a [support thread](support@lpcode.com.br)

This is young software. It works, but there's a lot left on the todo (check out the Other Notes tab). Have an idea? Let me know.
